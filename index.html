About Me

I'm an aspiring game and media developer studying at RIT in Rochester, NY. My goal is to create experiences that are truly authentic and immersive; games that can change the way you see the world around you. I aim to bring something to the table that's both fresh and familiar, that draws you into a state of flow and lets you bring yourself into the game world.

I love bringing innovative ideas into reality. I'm constantly teaching myself new skills and practicing to broaden my experiences so I always have the tools I need. I like to spend my free time playing indie gems, doing improv comedy, and vibing to alternative rock; but more than anything, I love meeting new people and hearing their stories!

Changeling VR: A Narrative Mystery

Description

Changeling is a narrative VR mystery game being developed in Unreal Engine. I worked on the project in Summer 2023, during which I was part of a team of 80+ developers. I worked with a focused team of 6 developers to renovation the gameplay and level design of a portion of the game.

Role: Interaction Designer

Responsibilities:

  • Developed gameplay for the Father/Dylan's level
  • Implemented new mechanics and refined existing interactions
  • Created a standardized system for snapping interactable objects together
  • Designed level space and progression
  • Wrote extensive documentation for the use of future teams

All-Wheel Brawl (In Development)

Description

All-Wheel Brawl is a local multiplayer arcade game I am currently developing with a team of 7. Our goal is to build the game on the PlayStation 5. We've prototyped the game in Unity using C# to flesh out the mechanics and system architecture, and we're now porting the project into a C++ project for the PS5. I've largely been responsible for designing gameplay mechanics and architecture for the Unity prototype, as well as assisting with sound design and mastering.

Roles: Gameplay Programmer/Designer, Sound Designer

Responsibilities:

  • Designing and implementing gameplay systems
  • Balancing parameters and abilities to support player experience
  • Mastering sound FX and background music
  • Porting Unity C# scripts to custom C++ engine
  • Implementing splitscreen multiplayer in both Unity and custom engine

Squadron Survival

Description

I created Squadron Survival using Unity and focused on the use of autonomous agents that can navigate a virtual environment. The agents make use of Reynolds' steering algorithms and non-intersection tests to steer towards their specified objectives.

Role: Sole Developer

Responsibilities:

  • Developed AI agents with various steering behaviors
  • Implemented gameplay mechanics and UI elements
  • Managed audio design and implementation
  • Designed animation state machines

BlackJack Web App

Description

My BlackJack Web App uses a Web API called DeckOfCards, which can be used to create decks of playing cards to shuffle and draw from. I used HTML and CSS to set up a webpage for the application, and I wrote a program in JavaScript that made the appropriate XHR requests to create decks and draw cards.

Role: Sole Developer

Responsibilities:

  • Created webpage using HTML and CSS
  • Implemented DeckOfCards Web API to handle decks and drawing
  • Designed JS event system for player input and API calls
